IgnorePointer 忽略事件，事件被传递到背后的视图了
AbsorbPointer 接收事件，并且自己把事件吃掉了
Container(
  width: 200,
  height: 200,
  color: Colors.blue,
  child: Stack(
    children: [
      Listener(
        onPointerDown: (event) {
          print("red down");
        },
        child: Container(color: Colors.red),
      ),
      Listener(
        onPointerDown: (event) {
          print("green down");
        },
        child: IgnorePointer(
          child: Container(color: Colors.green, width: 100, height: 100),
        ),
      ),
    ],
  ),
),
